网络数据挖掘项目文档¶

注意事项¶

本次项目使用的Python版本为3.6.5,推荐直接通过Anaconda进行安装，系统环境推荐使用基于*nix的系统，主要使用 到的Python包如下(粗体表示需要额外安装，之后可能还会有更新)：

  1. pandas
  2. numpy
  3. pickle
  4. gensim 安装
  5. jieba 安装
  6. faiss 安装
  7. surprise 安装

项目结构¶

说明¶

        DataAnalysis项目主要对数据进行预处理，包括新闻抽取，用户历史记录抽取过滤以及对新闻内容进行分词以建立其 TF-IDF向量等工作，同时该项目下存储了原始数据以及算法生成的中间数据。

        Recommend项目主要包括各种推荐算法的实现与封装，目前已可以正常使用的算法有基于内容推荐算法(topic_based_algo.py)，协同过滤推荐算法(collaborate_based_algo.py)以及一个 随机推荐算法(baseline.py)。同时该项目下包含一个工具类(utils.py)用于对算法在本次数据集上的性能进行评测，目前提供召回率与精确度两种度量方式，其定义如下：
